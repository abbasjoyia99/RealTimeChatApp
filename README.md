# Features
-Facebook Log In\
-Google Sign In\
-Email/Pass Registration / Log In\
-Photo Messages\
-Video Messages\
-Real Time Conversations\
-Location Messages\
-Search for Users\
-Deleting Conversations\
-User Profile\
-Dark Mode Support

# Set Up
After cloning the repo, you'll need to run pod install to install cocoapod dependencies.
